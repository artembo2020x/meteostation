MeteostationController - реализует CRUD-операции дя работы с метеостанциями.За основу берется модель Meteostation
MeteoController - собирает и выводит информацию по счетчикам различных метеостанций, предоставляет конечную информацию
SiteController - стандартный контроллер, предоставляемый фреймворком по умолчанию
